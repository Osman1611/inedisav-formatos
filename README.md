# INEDISAV · Formatos Académicos 2026

Sistema de gestión de formatos institucionales para IED Sarid Arteta de Vásquez (INEDISAV), Barranquilla.

## Formatos incluidos
- **Comisiones** — Estados de comisión por estudiante con contadores en tiempo real
- **Asistencia y Notas** — Grilla semanal + notas por periodo y promedio automático
- **Entrega de Informes** — Registro de asistencia de padres a reuniones

## Tecnología
- React 18
- Create React App
- localStorage para persistencia de datos

## Actualización anual
1. Exportar SIMAT del nuevo año como `.xlsx`
2. Actualizar `src/App.js` con los nuevos datos
3. Push a GitHub → Vercel despliega automáticamente

## Despliegue
Conectado a Vercel para despliegue automático en cada push.
